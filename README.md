何为设计模式？
  每一个模式描述了一个在我们周围不断发生的问题，以及该问题的解决方案的核心。这样你就能一次又一次的使用该方案而不必做重复的劳动。
  描述为：在一定环境中解决某一问题的方案，包括三个基本元素----问题，解决方案和环境。

模式分为三种类型：
  创建模式：可以简单的理解为，使创建对象的方式看起来更牛逼，更高大上的模式。
